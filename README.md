# e-commerce-shopping-portal-
🛒 E-Commerce Shopping Portal
Project Overview

The E-Commerce Shopping Portal is a responsive web-based shopping application developed using HTML, CSS, and JavaScript. The platform simulates an online shopping experience where users can browse products, add items to a cart, apply premium membership discounts, and generate invoices with real-time currency conversion using an external API.

The application demonstrates front-end web development concepts, API integration, inventory management, shopping cart functionality, and dynamic invoice generation.
